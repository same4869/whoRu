# 儿童钢琴启蒙扫盲帖：从业十年钢琴老师的经验之谈

## 学琴之前，这些事你要知道

无论家长抱着什么样的心态让孩子学习乐器，都要明白“万事开头难”的道理。很多家长希望通过学习钢琴来提升孩子的气质，但气质的培养并非一朝一夕之功，而是蕴藏在日常生活中的点滴积累。

孩子们天生对声音敏感，如果后天缺乏基本的、自然的、幸福的音乐教育，那么在正式学习乐器时，教师也难以顺利开展音乐教育，只能局限在认音、认节奏、认谱等基础内容。那么，如何在家庭中进行音乐教育呢？这是一个很大的话题，如果大家感兴趣，我很乐意继续分享。

对于没有音乐教育基础的家庭来说，在孩子正式开始上器乐课时，需要做些什么才能让这个过程更加甜蜜呢？下面我以钢琴为例，为大家讲解。

### 1. 如何挑选合适的启蒙老师？

教育的关键在于教育者本身的素质。我认为启蒙老师一定要“有趣”。

现在孩子们乐器启蒙的年龄基本在幼儿园中班下学期左右，这个年龄段的孩子注意力集中时间不超过十分钟，因此，课堂上适当安排音乐小游戏非常必要。试课时，如果老师能把钢琴当成大玩具，用高低音对应不同的小动物，让孩子觉得是在玩音色，那么这样的老师就可以考虑。

启蒙阶段一定要多试课，好的启蒙老师非常抢手。弹得一手好琴不一定教得好，但能教好琴的老师一定弹得不差。试课时，愿意和孩子分享自己喜爱曲目的老师也值得考虑。

好的老师总是稀缺的，课费也不会低。那些一周三次课，还完全不需要家长陪同的机构，其实和托管没有区别。如果家里有能带孩子放学后强身健体、保护视力的长辈，就别花这个冤枉钱。

钢琴老师也各有专长，大学教授能培养演奏家，但不一定能教好小宝宝。最重要的是“观察”，选择一位能陪伴孩子至少3-5年的启蒙老师，切忌选择机构托管。

### 2. 家长要不要陪练？怎么陪？

任何一种乐器教育都与音乐教育融为一体。在启蒙阶段，孩子不可能仅仅依靠任课教师每周一小时左右的讲解和示范就能完全掌握。

大部分家长都会选择陪练，称职的陪练非常重要。如果家长完全不陪练，孩子在家练习很可能原地踏步甚至倒退，导致下次上课时老师还在重复讲解相同的内容。有些家长觉得课费贵，喜欢请假，但事实上，无论请假多久，之后的几次课都会变着法儿复习前面的内容，久而久之形成恶性循环，最终钢琴变成家里最贵的摆设，浪费了孩子宝贵的时间。

那么，如何做好陪课陪练呢？

首先，家长自己要发自内心地喜欢音乐，或者想要成为孩子的榜样。有了这份原动力，才会愿意认真听老师的授课内容，做好课堂笔记，在家也会和孩子一起练习。

### 3. 短期内要不要考级？

考级的初衷是好的，但目前来看，它和其他教育类考级一样，已经变味了。考级曲目的难度与级别并不完全匹配，为了让孩子们都能考上，甚至出现三级水平的孩子也能弹奏六级曲目的情况。

我个人建议，自己教出来的孩子前三年不要参加任何考级。对于从别处转来的学生，我也会建议他们到最后一年直接考十级，不要浪费孩子宝贵的学琴时间。

相比之下，我更支持参加比赛。小比赛人人有奖，家家开心；大比赛则能体现实力，丰富阅历。

### 4. 别着急，乐曲也需要灵魂

家长们在启蒙课上经常会说，某首曲子太美了，如果孩子能学会就好了。但这类曲目为了迎合初学者，通常会改编成简易版本，缺少和声和音乐色彩。孩子还在三级水平，是弹不出六级曲目的效果的。

简易版已经让曲子失色不少，再加上孩子还处于初级阶段，处理曲目的能力还不够。所有的教育都是细水长流，只有夯实基础才能走得更远。
