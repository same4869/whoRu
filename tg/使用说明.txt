========================================
Telegram频道爬虫 - 使用说明
========================================

📁 项目结构：
tg/
├── config.py                     # API配置文件（需要自己填写）
├── tg_channel_scraper.py        # 方案1：API方式爬虫（需要API key）
├── web_scraper.py               # 方案2：网页爬虫（无需API，立即可用）
├── requirements.txt             # 依赖包列表
├── README.md                    # 详细说明文档
├── 快速开始.md                  # 快速入门指南
├── output/                      # 输出目录（爬取结果保存在这里）
│   ├── *.json                   # JSON格式数据
│   └── *.md                     # Markdown格式文档
└── .gitignore                   # Git忽略文件

========================================
✅ 快速使用（推荐 - 无需API）
========================================

1. 从项目根目录运行：
   python .\tg\web_scraper.py

2. 文件会自动保存到：tg/output/ 目录

3. 自定义要爬取的频道：
   - 编辑 tg\web_scraper.py 文件
   - 找到 main() 函数
   - 修改频道名称即可

========================================
⚠️ 重要说明
========================================

1. 网页爬取方式（web_scraper.py）：
   ✅ 优点：无需API，立即可用
   ❌ 缺点：只能获取最近20条左右消息

2. API方式（tg_channel_scraper.py）：
   ✅ 优点：可获取完整历史消息
   ❌ 缺点：需要申请API credentials

3. 输出文件位置：
   ✅ 正确：tg/output/
   ❌ 错误：output/（根目录）

========================================
🎯 如何爬取其他频道
========================================

方法1：修改代码
编辑 tg\web_scraper.py，找到：

def main():
    scraper = TelegramWebScraper()
    scraper.scrape_channel('这里改成你要的频道名', output_dir='tg/output')

方法2：批量爬取多个频道
修改为：

def main():
    scraper = TelegramWebScraper()
    channels = [
        'jmhbzsk',
        'telegram',
        'your_channel_here',
    ]
    scraper.scrape_multiple_channels(channels, output_dir='tg/output')
